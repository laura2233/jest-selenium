# jest-selenium
Jest + Selenium Test Plan Project

# Dependencies
- https://nodejs.org/en/download/
``` 
npm install selenium-webdriver
```
```
npm install --save-dev jest
```
```
npm install chromedriver
```

# Run
```
npm run test
```
